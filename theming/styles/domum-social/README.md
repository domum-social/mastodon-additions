# Domum.social Theme

A Mastodon custom theme with a warm + private aesthetic.

## Features
- Dark and light modes.
- Shared SCSS palette in `_domum-variables.scss`.
- Brand colors:
  - Golden Glow (#FFD86B)
  - Friendly Violet (#7D6BFF)
  - Teal Action (#34D1BF)
  - Dark Charcoal Base (#1C1F26)

## Installation
1. Copy this folder into:
app/javascript/styles/themes/domum-social/
